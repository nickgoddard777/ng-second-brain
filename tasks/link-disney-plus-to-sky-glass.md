# Link Disney+ Account to Sky Glass

Link the Disney+ account to Sky account on Sky Glass TV.

## Steps
- Press Sky button → Apps → Disney+
- Select "Link your account" — note the 4-digit code
- On phone/PC go to disneyplus.com/begin
- Sign in to Disney+ account and enter the 4-digit code
